# dotfiles

![screenshot](https://raw.githubusercontent.com/wiki/reviry/dotfiles/images/screenshot.png)

## Features

- **OS X** El Capitan 10.11.5
- **Terminal.app**
- **Tmux** 2.2
- **Zsh** 5.2
- **Vim** 7.4

## Installation

```
$ git clone https://github.com/reviry/dotfiles.git ~/dotfiles
$ cd ~/dotfiles && sh install.sh
```
